⚠️ Vos données ont été conservées

Bonne nouvelle ! Si vous aviez exporté vos données avant la désinstallation, vous pouvez les réimporter à tout moment en réinstallant l'extension.

  • Recherchez un fichier tawia-backup-*.json dans vos téléchargements
  • Si vous avez activé les backups automatiques, un fichier récent existe peut-être
  • Les données synchronisées avec votre compte Google peuvent être récupérées

🔄 Récupérer vos données

Pour retrouver vos intrants et projets :

1 Réinstallez Tawia depuis le Chrome Web Store
2 Ouvrez les Paramètres (⚙️) → onglet "Données"
3 Cliquez sur "Importer" et sélectionnez votre fichier de backup
